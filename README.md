## [INTERACTIVE-TIMELINE Live Demo](https://interactive-timeline-xi.vercel.app/)

Интерактивный блок с временной шкалой и слайдерами событий, реализованный в рамках тестового задания.

Блок отображает несколько временных отрезков (от 2 до 6).  
Для каждого временного отрезка выводятся:

- категория на круговой шкале;
- информация по временному отрезку;
- слайдер с подробными событиями, относящимися к активному отрезку времени.

При переключении временных отрезков:

- обновляются числа на окружности и контент под окружностью;
- появляется соответствующий слайдер с событиями выбранного отрезка;
- анимации переключения выполнены с помощью `gsap`.

Таймлайн блок (блок с временной шкалой + слайдер событий) реализован так, чтобы быть **полностью независимым**: если разместить на странице два и более таких блока, их верстка и логика не конфликтуют.

Используемые технологии выбраны в соответствии с требованиями тестового задания:

- **TypeScript**
- **React.js**
- **Webpack** — сборка проекта
- **styled-components** — стилизация
- **Swiper** — реализация слайдеров:
  - внешний слайдер категорий/временных отрезков;
  - внутренний слайдер событий внутри выбранного отрезка.
- **GSAP** (`gsap`, `@gsap/react`) — анимация смены слайдов и элементов интерфейса

## Инструкция по запуску проекта

```sh
$ git clone https://github.com/asenka0301/interactive-timeline.git
$ cd interactive-timeline
$ npm i
$ npm run build
$ npm run start
```
